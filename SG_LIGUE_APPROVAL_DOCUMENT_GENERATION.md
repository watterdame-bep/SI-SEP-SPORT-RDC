# Update: SG Ligue Approval - Document Generation & Email Routing

**Date**: March 4, 2026  
**Status**: ✅ COMPLETED  
**Changes**: 
1. Rejection emails now send ONLY to ligue (not federation)
2. Approval generates "Attestation d'Homologation de Ligue" document

---

## Problem Statement

### Issue 1: Rejection Email Recipients
Previously, rejection emails were sent to both federation and ligue. User requirement: send ONLY to the ligue.

### Issue 2: Missing Approval Document
When SG approves a ligue, no document was generated. User requirement: generate an "Attestation d'Homologation de Ligue" or "Acte de Reconnaissance" (not an Arrêté).

---

## Solution Implemented

### 1. Updated Email Routing

#### Rejection Email
**Before**: Sent to federation + ligue  
**After**: Sent ONLY to ligue

```python
# Envoyer email de rejet à la ligue uniquement
try:
    if ligue.email_officiel:
        subject = f"Rejet de la Ligue Provinciale - {ligue.nom_officiel}"
        context = {
            'ligue': ligue,
            'federation': ligue.institution_tutelle,
            'observations': observations_sg,
            'decision': 'REJETEE'
        }
        html_message = render_to_string('emails/ligue_decision.html', context)
        send_mail(
            subject,
            f"La ligue {ligue.nom_officiel} a été rejetée.",
            'noreply@sisep-sport.rdc',
            [ligue.email_officiel],  # ONLY ligue
            html_message=html_message,
            fail_silently=True
        )
except Exception as e:
    print(f"Erreur lors de l'envoi d'email: {e}")
```

#### Approval Email
**Status**: Unchanged - still sent to both federation and ligue

### 2. Created Attestation Document Generator

#### New Function: `generer_attestation_homologation_ligue()`

**Location**: `gouvernance/certificate_generator.py`

**Features**:
- Generates PDF document titled "ATTESTATION D'HOMOLOGATION DE LIGUE"
- Includes:
  - RDC government header
  - Secrétariat Général signature
  - Ligue information (name, sigle, province)
  - Affiliated federation
  - Validation date
  - Attestation number
  - QR code for verification
  - Signature space for SG
  - Professional formatting with RDC colors

**Document Structure**:

### 3. Updated Approval View

#### `sg_approuver_ligue()` Changes

**Added Document Generation**:
```python
# Générer le document d'attestation d'homologation
try:
    from gouvernance.certificate_generator import generer_attestation_homologation_ligue
    attestation_pdf = generer_attestation_homologation_ligue(ligue)
    
    # Sauvegarder le PDF dans le dossier media
    from django.core.files.storage import default_storage
    pdf_filename = f"institutions/attestations/Attestation_Homologation_Ligue_{numero_attestation}.pdf"
    attestation.document_attestation = default_storage.save(pdf_filename, attestation_pdf)
    attestation.save()
except Exception as e:
    print(f"Erreur lors de la génération du document d'attestation: {e}")
```

**Workflow**:
1. SG approves ligue
2. Attestation number generated
3. Attestation document created
4. PDF saved to media folder
5. Document path stored in attestation model
6. Email sent to federation + ligue
